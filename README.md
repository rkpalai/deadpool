# Xola Checkout Test Pages

This repository contains test pages for Xola checkout integrations.

## Available Test Pages

### index.html (Deadpool)
The original test page with "deadpool" branding. Contains various Xola checkout widget configurations for testing purposes.

**URL**: `index.html`

### xolaBot.html
A cloned version of the test page with "xolaBot" branding. Contains the same Xola checkout widget configurations but rebranded.

**URL**: `xolaBot.html`

## Features

Both test pages include:
- Multiple checkout button configurations
- Gift card widgets
- Embedded checkout experiences
- Various integration methods (buttons, divs, custom elements)
- Analytics integrations (Google Analytics, TikTok Pixel)

## Usage

Open either HTML file in a web browser to test Xola checkout integrations. The pages connect to Xola's staging environment for testing purposes.
